# 语言性能对比测试计划

## 目标
创建C、C++、Rust、Go、C#和Java六种语言的程序，测量它们执行1000万次加法的耗时，并进行对比。

## 实施步骤

### 1. 创建目录结构
为每种语言创建独立的目录：
```
lang_vs/
├── c/
├── cpp/
├── rust/
├── go/
├── csharp/
└── java/
```

### 2. 编写测试程序
为每种语言编写一个简单的程序，执行以下操作：
- 初始化一个计数器变量
- 记录开始时间
- 执行1000万次加法操作
- 记录结束时间
- 计算并输出耗时

### 3. 编写编译和运行脚本
创建一个统一的脚本`run_all.sh`，用于：
- 编译所有语言的程序
- 依次运行每个程序并记录输出
- 整理结果进行对比

### 4. 测试程序设计
所有程序将使用相似的算法：
```
int count = 0;
start_time = get_current_time();
for (int i = 0; i < 10000000; i++) {
    count++;
}
end_time = get_current_time();
elapsed = end_time - start_time;
print(elapsed);
```

### 5. 编译命令
- **C**: `gcc -O2 -o test test.c`
- **C++**: `g++ -O2 -o test test.cpp`
- **Rust**: `rustc -O test.rs`
- **Go**: `go build -o test test.go`
- **C#**: `dotnet build -c Release`
- **Java**: `javac Test.java`

### 6. 运行命令
- **C**: `./test`
- **C++**: `./test`
- **Rust**: `./test`
- **Go**: `./test`
- **C#**: `dotnet run -c Release`
- **Java**: `java Test`

## 预期结果
通过运行`./run_all.sh`脚本，将得到每种语言执行1000万次加法的耗时，便于进行直观对比。
