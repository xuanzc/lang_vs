# Python语言队列操作性能测试计划

## 目标
为现有性能测试框架添加Python语言支持，实现与其他语言相同的队列操作性能测试：
1. 创建长度为1000的字符型队列
2. 插入随机字符（A-Z, a-z, 0-9）
3. 队列长度超过999时自动出队
4. 循环执行10000次上述操作，计算总耗时

## 具体实现

### Python程序结构
```python
import time
import random
from collections import deque

LOOP_COUNT = 10000

# 生成随机字符（A-Z, a-z, 0-9）
def get_random_char():
    charset = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789"
    return random.choice(charset)

# 主测试函数
def main():
    q = deque(maxlen=1000)  # 使用固定长度的deque
    start_time = time.perf_counter()
    
    for _ in range(LOOP_COUNT):
        # 生成随机字符并入队
        c = get_random_char()
        q.append(c)
        
        # 由于使用了maxlen=1000，deque会自动出队，无需手动检查
    
    end_time = time.perf_counter()
    elapsed = (end_time - start_time) * 1000  # 转换为毫秒
    print(f"Python: {elapsed:.6f} milliseconds")

if __name__ == "__main__":
    main()
```
